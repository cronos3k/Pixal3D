Advanced Engine

How strongly the voxel skeleton is steered toward your image. 7–8 is a reliable starting point. Very high values can over-constrain and cause artefacts; very low values drift from the source.

Diffusion steps for the coarse 3D voxel skeleton (Sparse Structure stage). 12–20 gives good results. Above 25 the improvement is marginal; use 50 only for final hero renders.

Guidance scale for the SLaT shape refinement pass. Higher values lock surface features more tightly to image edges. Reduce slightly if you see over-sharpening or jagged edges.

Diffusion steps for high-res geometry refinement. 20 is a solid default. Push to 30–50 for production assets where you need maximum surface fidelity. Each step adds ~1–2 s on the A6000.

Diffusion steps for baking the PBR material (base colour, metallic, roughness). 20 is good for previews. Use 30–50 for final exports where texture quality matters most.

Size of the single UV texture atlas covering the entire mesh. 8192 is recommended when VRAM allows. Output file size grows roughly 4× with each step up. Choose 4096 for real-time game engines.

Hard upper limit on output faces (triangles). This is a simplification ceiling — it cannot create geometry. In Full Fidelity mode the model's natural face count is preserved; set lower only if you need a specific poly budget (e.g. 10000 for real-time).

Full Fidelity: keeps the exact mesh the model decoded, with per-voxel learned vertex offsets at sub-voxel precision. Recommended — this is the actual neural network output. DC Remesh: discards the decoded mesh and rebuilds topology using Dual Contouring at the voxel grid resolution (96 cells). Produces cleaner quad-dominant topology but loses the model's sub-voxel geometric detail and creates large flat facets in smooth areas.
